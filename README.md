# 👗 Fashion Trends Analysis

## 📊 Project Overview
This project analyzes fashion product data to understand pricing, discounts, and category trends.

## 🔍 Key Insights
- Most products are in budget and mid-range
- Topwear and Westernwear dominate
- Brands offer moderate to high discounts

## 📈 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit

## 🚀 Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Visualizations
- Insights Generation
